Visitor Log
1. Purpose

The purpose of the Visitor Log feature is to display and manage the list of all registered visitors in the system. It helps staff monitor visitor activity and review past visit records.

2. Expected User

The expected users of this feature are:

Security personnel

Receptionist

System administrator

They use the visitor log to view and track visitor records.

3. Main Functionality

The Visitor Log feature displays a table of registered visitors in the system.

The table includes the following information:

Visitor Name

Address

Contact Number

Purpose of Visit

Person to Visit

Date of Visit

Time In

Time Out

The page also allows the user to:

Search visitors by name

Filter visitors by date

View all recorded visitor entries

4. Acceptance Criteria

The system displays a table of all registered visitors.

Each visitor record shows the following details:
name, address, contact number, purpose of visit, person to visit, date, time in, and time out.

The system allows the user to search visitors by name.

The system allows the user to filter visitor records by date.

The visitor list updates automatically when new visitors are registered.
